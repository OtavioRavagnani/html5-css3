conteudo HTML:
<h1>Exemplo de titulo</h1>

<h1> = abertura de tag
</h1> = fechamento de tag
*Exemplo de titulo* = conteudo 

<p>exemplo de parágrafo</p>

<p> = abertura de tag 
</p> = fechadura de tag 
*exemplo de parágrafo* = conteudo 

<img src='foto.png' alt= 'Exemplo de foto'>

'<img src=' = abertura de tag 
'foto.png'= valor 
'alt=' = parâmetro 
'exemplo de foto = valor

Estilo em CSS:
 
h1 {
    font-family:Arial;
    font-size:20pt;
    color:blue;
}


h1{} = seletor 
 font-family = propriedade 
 font-size = propriedade 
 color = propriedade 
 'arial;'= valor
 '20pt;' = valor 
 'blue;' = valor 
  propriedade + valor = declaração
